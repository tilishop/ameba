    class Ameba::Config

    • Ameba::Config
    • Reference
    • Object

    Overview

    A configuration entry for Ameba::Runner.

    Config can be loaded from configuration YAML file and adjusted.

    config = Config.load
config.formatter = my_formatter

    By default config loads .ameba.yml file in a current directory.
